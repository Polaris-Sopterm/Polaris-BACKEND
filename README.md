## Quick Start
----
- `node` : 14.17.3

- `npm` : 6.14.13


- 의존성 설치
```
git clone https://github.com/Polaris-Sopterm/Polaris-BACKEND.git
cd Polaris-Backend
npm install
```

- db 연결
```
docker-compose up
```

- 로컬 서버 시작
```
NODE_ENV=ci npm run start
```

- dev 서버
```
https://polaris-dev-n3c7rawkea-du.a.run.app
```
